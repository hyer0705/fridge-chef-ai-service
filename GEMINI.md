# GEMINI.md (한국어)

## 프로젝트 개요

이 프로젝트는 "냉장고 셰프 AI 서비스"라는 웹 애플리케이션으로, 사용자가 가지고 있는 재료를 기반으로 레시피를 찾을 수 있도록 도와줍니다. 사용자는 재료 목록, 기분, 날씨를 입력하여 레시피를 제안받을 수 있습니다.

이 프로젝트는 다음 기술을 사용하여 구축되었습니다:

*   **Vite:** 최신 웹 프로젝트를 위한 빠른 빌드 도구입니다.
*   **React:** 사용자 인터페이스를 구축하기 위한 JavaScript 라이브러리입니다.
*   **TypeScript:** JavaScript의 상위 집합으로, 정적 타입을 지원합니다.
*   **Tailwind CSS:** 유틸리티 우선 CSS 프레임워크입니다.
*   **shadcn-ui:** 재사용 가능한 컴포넌트 모음입니다.

애플리케이션은 라우팅을 처리하는 기본 `App.tsx` 파일과 `src/pages/Index.tsx`에 위치한 메인 페이지로 구성되어 있습니다. UI는 재료 추가를 위한 `IngredientInput.tsx`와 레시피 표시를 위한 `RecipeCard.tsx` 등 여러 컴포넌트로 구축되었습니다.

## 빌드 및 실행

이 프로젝트를 빌드하고 실행하려면 Node.js와 npm이 설치되어 있어야 합니다.

1.  **의존성 설치:**

    ```bash
    npm install
    ```

2.  **개발 서버 실행:**

    ```bash
    npm run dev
    ```

    애플리케이션은 `http://localhost:8080`에서 확인할 수 있습니다.

3.  **프로덕션 빌드:**

    ```bash
    npm run build
    ```

## 개발 컨벤션

*   **스타일링:** 이 프로젝트는 스타일링을 위해 Tailwind CSS를 사용합니다.
*   **컴포넌트:** UI는 React 컴포넌트로 구축되었으며, 일부 기본 컴포넌트에는 shadcn-ui를 사용합니다.
*   **상태 관리:** 컴포넌트 수준의 상태 관리는 `useState`를 사용하고, 비동기 데이터 관리는 `react-query`를 사용합니다.
*   **라우팅:** 라우팅에는 `react-router-dom`을 사용합니다.
*   **린팅:** 린팅에는 ESLint를 사용하며, `npm run lint` 명령어로 린터를 실행할 수 있습니다.
