# 🍳 냉장고 속 식재료로 뭐 해먹지?

> 냉장고 속 잠자고 있는 재료들을 깨워 멋진 요리를 만들어보세요!

`냉장고 속 식재료로 뭐 해먹지?`는 냉장고 속 식재료를 활용하여 만들 수 있는 레시피를 추천해주는 웹 애플리케이션입니다. 이를 통해 낭비되는 식재료를 줄이고, 요리의 즐거움을 더해줍니다.

## ✨ 주요 기능

- **AI 기반 레시피 추천**: 보유한 재료, 날씨, 원하는 음식 스타일, 인원 수를 입력하면 AI가 맞춤 레시피를 추천해줍니다.
- **다양한 레시피 제공**: 한 가지 재료 조합으로 3가지 요리법을 제안하여 선택의 폭을 넓혀줍니다.
- **상세한 요리 정보**: 예상 조리 시간, 난이도, 영양 정보, 단계별 조리법 등 상세한 정보를 제공하여 요리 초보자도 쉽게 따라할 수 있습니다.

## 🛠️ 기술 스택

- **Framework**: React, Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS, shadcn-ui
- **State Management**: React Query
- **Routing**: React Router

## 🚀 설치 및 실행 방법

1.  **리포지토리 클론**
    ```bash
    git clone https://github.com/hyer0705/fridge-chef-ai-service.git
    ```
2.  **프로젝트 디렉토리로 이동**
    ```bash
    cd fridge-chef-ai-service
    ```
3.  **의존성 설치**
    ```bash
    npm install
    ```
4.  **개발 서버 실행**
    ```bash
    npm run dev
    ```
    서버가 실행되면 `http://localhost:8080` 에서 서비스를 확인할 수 있습니다.
